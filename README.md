# Study Brain

An AI-assisted study tool built with Python and Flask.

## Features

- Upload notes and images
- OCR text extraction
- Automatic summaries
- Keyword extraction
- Local processing (no cloud required)

## Tech Stack

- Python
- Flask
- Tesseract OCR
- HTML

## Future Plans

- Revision questions
- Search notes
- Flashcards
- Local AI integration
